## How to Use

Each project has its own directory with documentation. To get started:

```bash
cd <project-name>
cat README.md
```

For example, to run the Server Performance Stats project:

```bash
cd Server-Performance-Stats
chmod +x server-stats.sh
./server-stats.sh
```
